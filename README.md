# INFO
Lahka skripta za beleženje časovnega zamika med lokalnim (client) in centralnim (server) časom
# WHY
V izogib spreminjanju lokalnega časa in možnih zapletov pri logiranju kode
# HOW-TO
Spremeniti je treba sledeče zadeve:
* SERVER: naslov strežnika (najbolje kar IP)
* CSV_FILE: naslov datoteke, kamor se shranjujejo zapisi
* TIME_DELAY_SECONDS: časovni zamik (v sekundah) med klici na strežnik
